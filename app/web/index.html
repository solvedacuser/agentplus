Agent Overview

강의 PDF 기반 시험공부 코치 Agent

이 Agent는 사용자의 자연어 요청을 분석해 개념 설명, 예상문제 생성, 답안 채점 Tool을 선택하고 실행합니다. 개념 설명과 문제 생성은 먼저 PDF를 인덱싱해 만든 RAG 검색 결과를 근거로 사용합니다.

Tools

Agent가 실행하는 Tool

concept_explain_tool
PDF/RAG 검색 결과를 바탕으로 개념 설명, 핵심 포인트, 출처, 후속 질문을 반환합니다.
quiz_generate_tool
선택한 주제에 대해 객관식 또는 단답형 예상문제와 정답, 해설, 출처를 생성합니다.
answer_grade_tool
최근 생성된 문제와 세션 메모리를 기준으로 사용자 답안을 채점합니다. 한 문항 답안뿐 아니라 줄 단위로 번호를 붙인 다문항 답안도 한 번에 처리합니다.

답안 채점

Multi-question Grading

답안 채점은 다문항 입력도 가능합니다

입력 형식

답안 채점:
1번: 첫 번째 문제에 대한 내 답안
2번: 두 번째 문제에 대한 내 답안
3번: 세 번째 문제에 대한 내 답안

동작 방식

  • 최근 생성된 quiz의 1번, 2번, 3번 문제와 사용자의 번호별 답안을 매칭합니다.
  • 각 문항마다 `answer_grade_tool`을 개별 호출해 정답 여부, 점수, 해설을 만듭니다.
  • 채점 결과는 세션 메모리에 저장되고 평균 점수와 정답 개수가 함께 요약됩니다.
  • 번호 없이 입력하면 최근 quiz의 1번 문제에 대한 단일 답안으로 처리됩니다.

Execution Flow

  4. Create Embeddings
  5. Save Vector Store
  6. User Prompt
  7. classify_request
  8. Retrieve Context
  9. Run Tool
  10. Save Session Memory
  11. Return Response
